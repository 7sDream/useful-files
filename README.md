It can disable your keyboard, when after run it, you will get another script to enable keyboard.

```bash

```

Open Wifi AP.

Need wlan0 suppose ap mode.

Need hostapd & dnsmasq & nmcli

```bash
sudo apt-get install hostapd dnsmasq
```
